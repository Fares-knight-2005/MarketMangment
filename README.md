# MarketMangment

A minimalist Supermarket Management System backend built with .NET, featuring basic CRUD operations for inventory and product tracking.

## 🚀 Features

*   **Inventory Management**: Easily track product stock levels.
*   **CRUD Operations**: Create, Read, Update, and Delete functionality for supermarket items.
*   **Lightweight**: Built with a focus on simplicity and efficiency.
